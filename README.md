## Usage example

```bat
> cls && set RUST_LOG=duckdns_rs && set DUCKDNS_TOKEN=<TOKEN-GOES-HERE> && set DUCKDNS_DOMAIN=<DOMAIN-GOES-HERE> && duckdns-rs.exe

```
